# Personal Portfolio Website

A simple and responsive personal portfolio website built using HTML and CSS.

## Overview

This project showcases my personal profile, skills, and contact information through a clean and modern user interface. It is designed to serve as an online portfolio and learning project for front-end web development.

## Features

* Responsive design
* Modern UI with custom styling
* Animated gradient background
* Contact section
* Social media links
* Built using pure HTML and CSS

## Technologies Used

* HTML5
* CSS3

## Project Structure

```
portfolio/
│
├── index.html
├── contact.html
├── style.css
└── README.md
```

## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/prabhhhuu/portfolio-simple-website.git
```

2. Open the project folder.

3. Launch `index.html` in your browser.

## Customization

You can easily customize:

* Name and profile information
* Colors and theme
* Social media links
* Contact section
* Fonts and animations
